# Audiencemeter
Um registrador de audiência que apresenta estatísticas sobre a presença dos participantes

Uma certa reunião ocorre toda semana. A assistência é dividida em alguns grupos e um integrante de cada grupo é responsável por anotar quais membros estavam ausentes e quantos vieram. Todos esses dados então são reunidos para descrever a situação da assistência total

Cada participante do grupo vai estar cadastrado, sem informações pessoais além do nome. As faltas e presenças se dariam por meio desses cadastros que o responsável pela contagem tem acesso

Um mecanismo de autenticação e um campo para descrever o motivo da ausência de cada participante do grupo seria interessante

Talvez um alerta quando certo participante começasse a faltar muitas vezes seguidas

Quando alguém esquecesse de fazer a contagem, um alerta ou ícone denunciaria isso na página que todos têm acesso
